# BroadbandNow Brand Guidelines

## Brand Name
BroadbandNow — one word, lowercase in logo form ("broadbandnow"), title case in running text ("BroadbandNow"). Never "Broadband Now" or "BROADBANDNOW."

## Color Palette

### Primary
- **Crimson Bolt**: #DC2626 — primary brand color. Matches the lightning bolt in the logo. Used for CTAs, key accents, and brand marks.
- **Charcoal**: #1C1917 — primary dark. Logo text color, body text, dark backgrounds, navigation.

### Secondary
- **Slate Blue**: #3B82F6 — secondary accent. Used for links, hover states, secondary buttons, and informational elements.
- **Amber**: #F59E0B — urgency/attention color. Used sparingly for limited-time badges and promotional callouts.

### Neutrals
- **Stone 600**: #57534E — secondary text
- **Stone 300**: #D6D3D1 — borders, dividers
- **Stone 100**: #F5F5F4 — light backgrounds, alternating sections
- **White**: #FFFFFF — page background, card backgrounds

## Typography

### Heading Font
**Inter** — weights 600 (semibold) and 700 (bold). Use for all headings, CTAs, and navigation.
Source: Google Fonts

### Body Font
**Inter** — weights 400 (regular) and 500 (medium). Use for body text, descriptions, and form labels.
Source: Google Fonts

### Type Scale
- H1: 3.5rem / line-height 1.1 — page hero headline only
- H2: 2.25rem / line-height 1.2 — section headings
- H3: 1.5rem / line-height 1.3 — card headings, sub-sections
- Body: 1rem / line-height 1.6 — paragraphs, descriptions
- Small: 0.875rem / line-height 1.5 — legal text, footnotes, captions

## Logo
- File: `logo.png`
- Lowercase "broadbandnow" in Charcoal with a Crimson Bolt lightning bolt accent replacing the dot on "now"
- On dark backgrounds, text switches to white; lightning bolt stays Crimson Bolt
- Minimum width: 140px
- Clear space: equal to the height of the lightning bolt on all sides

## Tone of Voice
- **Confident, not aggressive**: "Get gigabit speed" not "Crush the competition"
- **Clear, not clever**: Straightforward language, no puns or jargon
- **Modern, not trendy**: Clean and current without chasing trends
- **Helpful, not pushy**: Guide the visitor, don't pressure them

### Words to use
fast, reliable, simple, transparent, included, guaranteed, fiber-powered

### Words to avoid
cheap, deal, limited time only (overused), revolutionary, game-changing, best-in-class

## Visual Style
- **Clean and spacious**: Generous whitespace, clear section separation
- **Bold and warm**: Charcoal + Crimson Bolt as the dominant palette, with warm stone neutrals
- **Rounded corners**: 8px for cards and containers, 6px for buttons, 4px for inputs
- **Shadows**: Subtle, diffused shadows (no hard drop shadows). Use for cards and elevated elements.
- **Icons**: Outlined style, 1.5px stroke, rounded caps. Use Slate Blue or Stone 600.
- **Photography**: Not used — rely on clean iconography, the bold color palette, and whitespace

## Do's and Don'ts

### Do
- Use Crimson Bolt for all primary CTAs
- Maintain generous padding between sections (min 80px desktop, 48px mobile)
- Use Charcoal for dark section backgrounds (not pure black)
- Keep forms short — zip/address field only for the availability check

### Don't
- Don't use Amber for CTAs — it's for urgency badges and alerts only
- Don't use more than 2 colors in a single section background
- Don't center-align body text — left-align always (headings can be centered)
- Don't use stock photography — stick to iconography and the brand color palette
